How many days has Colin Kaepernick been denied work in the NFL? We're counting: [@KapWatch](https://twitter.com/KapWatch)

---

This is a simple Twitter bot that takes a screenshot of [kapwatch.com](https://kapwatch.com), updates the Twitter account's profile picture, and tweets an image every morning. By [@jamilahking](https://twitter.com/jamilahking) and [@joshbegley](https://twitter.com/joshbegley).

Dependencies: `Tweepy` `CasperJS` `PhantomJS`
